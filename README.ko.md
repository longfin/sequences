# Sequences

[English](README.md) · **한국어** · [日本語](README.ja.md)

사진집 **시퀀싱**을 위한 로컬 우선 웹앱입니다. 레이아웃 소프트웨어를 열기
전에, 어떤 사진을 어디에 둘지 정하는 편집 작업을 브라우저에서 합니다. 사진을
트레이에 올리고, 스프레드(펼침면) 위로 드래그해 배치하고, 마주보는 쌍을
살피고, 더미북처럼 넘겨봅니다.

모든 데이터는 브라우저(IndexedDB)에만 저장됩니다. 서버도, 업로드도 없습니다.

**바로 써보기 → [sequences.kelupus.com](https://sequences.kelupus.com/)**

![더미북 프리뷰에서 스프레드 넘겨보기](docs/demo.gif)

## 왜 만들었나

InDesign 같은 조판 도구도 스프레드 뷰는 보여줍니다 — 다만 문서 설정, 마스터,
스타일처럼 아직 필요 없는 것들과 함께요. 시퀀싱 단계에서 원하는 건 그냥
사진을 페이지에 얹어놓고 바라보는 것뿐입니다. Sequences는 딱 그만큼입니다:
탭 하나 열고, 사진을 끌어다 놓고, 흐름이 맞을 때까지 섞어보기. 설치도 설정도
없습니다.

## 기능

- **스프레드 편집** — 책을 펼쳐놓은 것처럼 스프레드가 세로로 흐릅니다.
  트레이→페이지 드래그 배치, 페이지 간 교체, 스프레드 순서 변경. 페이지는
  verso/recto와 실제 페이지 번호로 표시됩니다.
- **페이지 프리셋** — 전면 재단 / 여백 레이아웃 / 빈 페이지(페이싱용).
  페이지 비율 선택 가능 (4:5, 3:4, 2:3, 1:1, 가로형).
- **흑백 토글** — 톤의 흐름 확인용.
- **오버뷰** — 전체 스프레드를 콘택트시트처럼 한 화면에.
- **넘겨보기** — 3D 책장 넘김 프리뷰. 편집 중이던 스프레드에서 시작합니다.
  방향키/스페이스로 넘김.
- **PDF 내보내기** — 스프레드당 한 페이지, 접지선과 폴리오 포함. 프린트
  더미북 제작용.
- **로컬 저장** — 사진과 시퀀스가 IndexedDB에 저장되어 새로고침해도
  유지됩니다. 내보내기는 원본 해상도, 화면은 생성된 썸네일 사용.
- **저장/불러오기/리셋** — 프로젝트 전체(시퀀스+사진)를 파일 하나로 저장하고
  다른 기기에서 불러오거나, 전부 지우고 새로 시작.
- **다국어** — 한국어 / English / 日本語 (자동 감지, 전환 가능).

## 실행

```bash
npm install

```

않습니다.
